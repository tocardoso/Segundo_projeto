# Descrição

Este script simples em Python tem como objetivo comparar dois números inteiros introduzidos pelo utilizador. O programa verifica se os valores são iguais ou diferentes e inclui um tratamento de erros para garantir que apenas números válidos sejam processados.

# Instruções

1.  Abre o terminal ou consola na pasta onde se encontra o ficheiro numeros_iguais.py
2.  Executa o comando: python numeros_iguais.py ou abre o ficheiro diretamente no VSCode, Spider e fazer Run.
3. De seguida, o progama vai apresentar:
   Introduz um número inteiro:
   Introduz outro número inteiro:
   Os números introduzidos são diferentes.(se os numeros introduzidos forem diferentes)
   Os números são iguais.(se os numeros introduzidos forem iguais)
